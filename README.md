# Monkey Interpreter

Implementation of an interpreter for the Monkey programming language following the book **"Writing an Interpreter in Go" (WAIIG)** by Thorsten Ball.

## About the Monkey Language

Monkey is a programming language created for educational purposes, with C-inspired syntax.
